# Ping Backend

Ping 프로젝트의 백엔드 배포용 저장소입니다.

## Overview
본 프로젝트는 팀 프로젝트로 시작되었으며,  
초기 개발 및 협업은 별도의 팀 저장소에서 진행되었습니다.

이후 배포 및 서버 운영을 위해  
백엔드만 분리하여 이 저장소에서 관리했습니다.

Express와 MySQL(MariaDB)을 기반으로  
게시글, 이미지, 질문, 답변, 카테고리 관련 API를 구성했습니다.

## Tech Stack
- Node.js
- Express
- MySQL / MariaDB

## Main Features
- 게시글 등록 / 조회 / 삭제 API
- 이미지 업로드 및 관리
- 질문 / 답변 CRUD
- 카테고리 데이터 처리
- 관리자 기능 관련 API
