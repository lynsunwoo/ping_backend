# Ping Backend

Ping 프로젝트의 백엔드 배포용 저장소입니다.

## Overview
기존 팀 프로젝트 저장소에서 배포 및 서버 운영을 위해  
백엔드만 별도로 분리하여 관리한 레포지토리입니다.

Express와 MySQL(MariaDB)을 기반으로  
게시글, 이미지, 질문, 답변, 카테고리 관련 API를 구성했습니다.

## Tech Stack
- Node.js
- Express
- MySQL / MariaDB

## Main Features
- 게시글 등록 / 조회 / 삭제 API
- 이미지 업로드 및 관리
- 질문 / 답변 CRUD
- 카테고리 데이터 처리
- 관리자 기능 관련 API
